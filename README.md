# 活点账单 - 微信小程序

一款AA制分账小程序，采用哈利波特魔法主题设计。

## 功能特性

- ⚡ **施咒** - 均分咒与清算咒，简单快速分账
- 🏦 **古灵阁** - 记账事件与账单管理，详尽的收支明细和清算契约
- 🧙 **休息室** - 巫师伙伴管理，快速召唤参与分账
- 📖 **冥想盆** - 历史账单记录查看，账单详情与清算明细
- 🎭 **30个巫师头像** - 精美头像，自动或手动分配
- 📱 **新手引导** - 悬浮式功能引导，快速上手

## 技术栈

- **前端框架**: Taro 4.x + React 18 + TypeScript
- **样式**: Sass
- **存储**: 本地存储
- **构建工具**: Webpack 5

## 开发指南

### 1. 安装依赖

```bash
cd magic-bill-mp
npm install
```

### 2. 开发模式

```bash
npm run dev:weapp
```

### 3. 打开微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `magic-bill-mp` 目录
3. 填入您的 AppID
4. 点击编译预览

## 项目结构

```
magic-bill-mp/
├── config/                 # Taro 配置
├── src/
│   ├── app.tsx             # 应用入口
│   ├── app.config.ts       # 应用配置
│   ├── app.scss            # 全局样式
│   ├── assets/             # 静态资源
│   ├── components/         # 公共组件
│   ├── constants/          # 常量配置
│   ├── pages/
│   │   ├── index/          # 施咒页
│   │   ├── ledger/         # 古灵阁
│   │   ├── companions/     # 休息室
│   │   ├── history/        # 冥想盆
│   │   ├── share/          # 分享页
│   │   ├── simple/         # 均分咒
│   │   └── multi/          # 清算咒
│   ├── services/           # 业务服务
│   ├── types/              # 类型定义
│   └── utils/              # 工具函数
├── dist/                   # 编译输出目录
└── project.config.json     # 小程序配置
```
