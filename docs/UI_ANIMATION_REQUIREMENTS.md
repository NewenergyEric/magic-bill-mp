# 魔法记账：功能、命名与动效全案

> 为了让这款微信小程序不仅仅是一个工具，而是一个让用户感到"身临其境"的魔法道具，这套方案结合了哈利·波特风格的叙事感与现代移动端的流畅体验。

---

## 一、全局视觉风格建议 (UI/UX)

### 1. 配色方案 (Palette)

| 用途 | 颜色 | 色值建议 |
|------|------|----------|
| 主色调 | 格兰芬多红 (Burgundy) | `#722F37` |
| 主色调 | 魔法金 (Old Gold) | `#CFB53B` |
| 背景色 | 羊皮纸色 (Parchment) | `#F5E6C8` |
| 背景色 | 深夜蓝 (Midnight Blue) | `#191970` |
| 强调色 | 吼叫信红 (异议) | `#8B0000` |
| 强调色 | 荧光闪烁绿 (成功同步) | `#39FF14` |

### 2. 字体选择

- **标题**：带有衬线的、略显古老的字体（如小程序可加载自定义 WebFont）
- **数字**：类似手写体或复古打字机字体

### 3. 触感反馈 (Haptic Feedback)

| 场景 | 效果 |
|------|------|
| 施咒成功 | 短促的双击震动 |
| 收到吼叫信 | 持续的、不规则的轻微震动 |

### 4. 音效设计 (Soundscape)

| 场景 | 音效 |
|------|------|
| 打开小程序 | 羊皮纸展开的沙沙声 |
| 金币结算 | 清脆的硬币碰撞声 (Clink!) |

## 二、功能与动效对照表

| 现实功能 | 魔法命名 | 交互与动效设计 | 沉浸感/心理暗示 |
|----------|----------|----------------|-----------------|
| 创建/加入群组 | **签署血缘契约 (Blood Pact)** | 屏幕出现一张展开的羊皮纸，用户点击"签署"时，指尖触碰处出现暗红色墨水扩散效果，随后契约卷起。 | 建立"不可违背"的信任感。 |
| 添加一笔账单 | **施展记账咒 (Spell Casting)** | 点击"+"号时，图标变为一根魔杖，划出一个"Z"字形闪电光效。账单输入框像烟雾一样聚拢而成。 | 让枯燥的录入变得像施法一样有趣。 |
| 多人均分账单 | **均分咒 (Aequitas)** | 屏幕中央的一大堆金加隆（硬币）随着咒语炸开，均匀地飞向四周代表不同成员的头像。 | 视觉化"公平"的过程。 |
| 账单异议/申诉 | **寄出吼叫信 (The Howler)** | 点击异议时，账单卡片变成一个红色的信封，信封长出嘴巴并剧烈抖动。音效：轻微的咆哮声。 | 幽默地化解催账/质疑的尴尬。 |
| 账单修改记录 | **冥想盆回溯 (Pensieve)** | 点击查看历史时，屏幕出现水面波纹涟漪，旧的账单数据从"水底"缓缓浮现，背景变为深蓝色。 | 强调"记忆不可篡改"的真实性。 |
| 离线记账状态 | **自动书写羽毛笔 (Self-Writing Quill)** | 在断网时，屏幕右上角出现一根自动书写的羽毛笔在沙沙作响。联网后，羽毛笔化为一道金光消失。 | 缓解用户对数据丢失的焦虑。 |
| 最终结算/清账 | **终极清算/魔法平衡 (Equilibrium)** | 两个巫师的债务对冲时，两道不同颜色的光束碰撞并抵消，最后化为漫天金色的纸屑。 | 带来债务清零的巨大快感。 |
| 导出Excel数据 | **提取魔法卷轴 (Scroll Extraction)** | 点击导出，屏幕中心出现一个旋转的漩涡，一卷长长的羊皮纸卷轴被缓缓拉出，最后变为微信文件图标。 | 将枯燥的数据导出仪式化。 |
| 成员退出契约 | **化为幽灵 (Ghosting)** | 退出的成员头像变灰并半透明化，像霍格沃茨的幽灵一样漂浮在列表末尾，直到账务结清。 | 既保留了数据，又符合魔法逻辑。 |

## 三、微信小程序特别优化

### 1. 订阅消息

当有人对你发出"吼叫信"或"清算提醒"时，微信通知的标题可以是：

> "您收到了一封来自[好友名]的吼叫信，请尽快查看！"

### 2. 分享卡片

分享到群聊的卡片设计成一张**"魔法邀请函"**，文案是：

> "[姓名]邀请你签署一份关于[活动名]的魔法契约，点击开启金库。"

## 五、技术实现备注

1. **动画实现**：使用 CSS Animation + Lottie（复杂动画）
2. **音效**：小程序 InnerAudioContext API
3. **震动**：Taro.vibrateShort / Taro.vibrateLong
4. **自定义字体**：小程序 loadFontFace API
