# 活点账单 v2.0.0 测试指南

## 测试文件位置

- `tests/README.md` - 测试用例说明文档
- `tests/unit.test.ts` - 单元测试代码

## 运行测试

```bash
# 安装依赖（如果尚未安装）
npm install

# 运行测试
npm test

# 或使用 vitest
npx vitest run
```
